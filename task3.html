3☁️ חיבור ל-Cloud וצפייה בלידים

עכשיו שיש לכם דף נחיתה עם טופס, הגיע הזמן לחבר אותו לבסיס נתונים כדי לשמור את הלידים ולראות אותם.

מה זה Lovable Cloud?

Lovable Cloud זה המקום שבו נשמר כל המידע מהאתר שלכם:

  • הפניות מהטפסים - כל הלידים במקום אחד
  • משתמשים - אם תרצו להוסיף התחברות
  • קבצים - תמונות ומסמכים

הכל עובד אוטומטית - פשוט אומרים ל-Lovable מה רוצים והוא מטפל בכל השאר!

1

משימה 1: הפעלת Cloud בפרויקט ☁️

שלב א' - פתיחת Cloud:

  1. לחצו על כפתור ה-+ שליד Preview
  2. בחרו Cloud

שלב ב' - אישור החיבור:

  • Lovable ישאל אם להפעיל Cloud לפרויקט
  • לחצו Allow לאישור
💡 טיפ: אפשר לשנות את ההגדרות ב-Settings → Account → Tools
2

משימה 2: חיבור הטופס לשמירת לידים 📊

בקשו מ-Lovable לשמור את הפניות מהטופס בבסיס הנתונים.

פרומפט לדוגמא:

כשמישהו שולח את טופס יצירת הקשר, שמור את הפרטים שלו בבסיס הנתונים. אחרי השליחה תופיע הודעת "תודה! נחזור אליך בהקדם".

זהו! Lovable יטפל בכל השאר - ייצור טבלה מתאימה וידאג שהמידע יישמר.

3

משימה 3: צפייה בלידים שנשמרו 👀

כאן הקסם קורה - אפשר לראות את כל הלידים שנכנסו!

איך נכנסים:

  1. לחצו על Cloud (כפתור ה-+ ליד Preview)
  2. בתפריט השמאלי, לחצו על Database
  3. תראו רשימה של כל הטבלאות
  4. לחצו על הטבלה leads

מה רואים:

  • כל השורות (לידים) שנשמרו
  • אפשר לסנן ולחפש
  • אפשר לערוך ערכים ישירות
  • אפשר למחוק שורות
💡 טיפ: אין צורך לכתוב SQL! הכל נעשה דרך ממשק ויזואלי פשוט

💡 טיפים חשובים

📌 חינם לנסות: כל Workspace מקבל שימוש חינמי בחודש - מספיק להרבה פרויקטים

📌 פשוט לתפעול: אין צורך בידע טכני - הכל דרך לחיצות וממשק פשוט

📌 עדכון מיידי: ברגע שמישהו שולח טופס - רואים את זה מיד ב-Cloud

📌 שימו לב: ברגע שמפעילים Cloud בפרויקט, אי אפשר לבטל את זה

🔧 משהו לא עובד?

הטופס לא שומר נתונים:

  • ודאו שה-Cloud מופעל (יש סימן ענן)
  • נסו לרענן את הדף

לא רואים את הלידים:

  • לחצו Refresh בממשק ה-Database
  • ודאו שהטופס נשלח בהצלחה

✅ רשימת בדיקה לפני סיום

  • Cloud מופעל בפרויקט
  • הטופס שומר פניות בבסיס הנתונים
  • אני יכול/ה לראות לידים ב-Cloud → Database
