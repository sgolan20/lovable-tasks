1🎨 עיצוב דף הנחיתה לצבעי המותג

עכשיו שיש לכם דף נחיתה עובד, הגיע הזמן להתאים אותו לזהות החזותית של המותג שלכם באמצעות פיצ'ר ה-Themes של Lovable.

🚀 איך נכנסים ל-Themes?

  1. לחצו על כפתור ה-+ שליד Preview
  2. בחרו Design view
  3. עברו ללשונית Themes בפאנל השמאלי
1

משימה 1: בחירת Theme בסיסי 🎯

נסו להחיל אחד מה-Templates המובנים של Lovable וראו איך הדף משתנה.

מה לעשות:

  • גללו בין ה-Themes המוכנים
  • לחצו על אחד מהם לתצוגה מקדימה
  • לחצו Apply כדי להחיל
2

משימה 2: יצירת Theme מותאם אישית 🎨

צרו Theme חדש עם צבעי המותג שלכם.

מה לעשות:

  • לחצו Create כדי ליצור Theme מהפרויקט הנוכחי
  • ערכו את הצבעים לפי המותג שלכם:
    • Primary Color - הצבע הראשי (כפתורים, קישורים)
    • Secondary Color - צבע משני
    • Background - צבע רקע
    • Text Color - צבע טקסט
💡 טיפ: אם יש לכם קובץ עיצוב, העתיקו משם את קודי ה-HEX (למשל #3182ce)
3

משימה 3: התאמת טיפוגרפיה ✍️

שנו את הפונטים כדי שיתאימו למותג.

אפשרות א': בחירה מהרשימה הקיימת

הדרך הפשוטה ביותר - בחרו פונט מתוך הרשימה המובנית:

  • בתוך ה-Theme, לחצו על Typography
  • בחרו פונט לכותרות (Headings)
  • בחרו פונט לטקסט רגיל (Body)
  • התאימו גדלים ומרווחים לפי הצורך

אפשרות ב': העלאת פונט מותאם אישית

אם יש לכם פונט ייחודי למותג:

  • גררו קובץ פונט (TTF או OTF) ישירות לתוך Lovable
  • כתבו פרומפט שמבקש להשתמש בפונט

פרומפט לדוגמא:

השתמש בפונט שהעלתי ככותרות ראשיות בכל הדף

אפשרות ג': שימוש ב-Google Fonts

פרומפט לדוגמא:

השתמש בפונט Heebo מ-Google Fonts לכל הטקסטים בעברית. ודא שהפונט נטען באמצעות useEffect React hook.

פונטים מומלצים לעברית:

Heebo
נקי ומודרני
Assistant
ידידותי וקריא
Rubik
מעוגל ונעים
Open Sans Hebrew
קלאסי ומקצועי
4

משימה 4: Visual Edits - עריכה ישירה על הדף 🖱️

השתמשו ב-Visual Edit לשינויים נקודתיים.

מה לעשות:

  • עברו ללשונית Visual edits
  • לחצו על אלמנט בדף (כפתור, כותרת, תמונה)
  • שנו צבע, טקסט, או מרווחים מהסרגל הצדדי
💡 טיפ: החזיקו Cmd/Ctrl כדי לבחור כמה אלמנטים יחד

💡 טיפים חשובים

📌 לא עולה קרדיטים! עריכה ויזואלית לא מנכה קרדיטים (חוץ מאלמנטים דינמיים)

📌 שמירה ושיתוף: ה-Theme נשמר ב-Workspace וזמין לכל הפרויקטים

📌 ייבוא CSS: אם יש לכם קובץ CSS קיים, אפשר לייבא אותו ישירות

📌 תצוגה מקדימה: כל שינוי נראה בזמן אמת לפני שמירה

✅ רשימת בדיקה לפני סיום

  • צבע ראשי מתאים למותג
  • צבע כפתורים בולט וקריא
  • פונטים נקיים וקריאים
  • ניגודיות טובה בין טקסט לרקע
  • העיצוב נראה טוב במובייל
